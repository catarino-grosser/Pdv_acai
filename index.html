Configuração do Firebase

Crie um projeto no Firebase, ative o Cloud Firestore e cole as configurações no bloco firebaseConfig dentro do código.
As chaves abaixo precisam ser substituídas pelas suas.
const firebaseConfig = {
  apiKey: "COLE_AQUI",
  authDomain: "COLE_AQUI",
  projectId: "COLE_AQUI",
  storageBucket: "COLE_AQUI",
  messagingSenderId: "COLE_AQUI",
  appId: "COLE_AQUI"
};
